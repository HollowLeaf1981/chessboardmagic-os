# Piece Trainer

Practise how chess pieces move by collecting targets on the board.

## Overview

Piece Trainer is an interactive movement trainer for queens, rooks, bishops, knights, and kings. The player chooses a piece and a mode, then starts a new game with the selected piece placed on a random square.

In Timer mode, the goal is to drag the piece to as many target pawns as possible before the 60-second timer runs out. Each captured target increases the score, and clearing all targets advances the level with a new set of targets.

In Fewest Moves mode, the player draws legal move arrows on the board to plan a route. Arrows can be toggled on or off by drawing the same move again, and the arrow list can be cleared at any time.

## Data

This app does not use an external dataset. Piece positions, target squares, levels, and move validation are generated locally.
