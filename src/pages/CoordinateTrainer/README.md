# Coordinate Trainer

Learn chessboard coordinates by clicking the square shown before the timer runs out.

## Overview

Coordinate Trainer is a fast board-recognition game. Each round starts with a short 3-2-1 countdown, then gives the player 60 seconds to identify as many target squares as possible. The target coordinate is shown in the centre of the board, and the player scores one point each time they click the correct square.

After each click, the app immediately selects a new random target square. The game can be stopped manually, and the highest score is saved locally in the browser.

## Data

This app does not use an external dataset. The board coordinates are generated locally from files A-H and ranks 8-1.

The board square styling comes from `@/config/chessboard`, so it uses the same shared board theme as the rest of the project.
